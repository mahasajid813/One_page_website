# 🌐 One-Page Website

This project is a responsive one-page website created using HTML, CSS, and JavaScript. The website demonstrates basic web development concepts including layout design and styling. 
---

## 🚀 Features

- Responsive one-page layout
- Image gallery with lightbox/modal
- Smooth styling and animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

```plaintext
One-Page Website/
│── one_page_website.html
│── CSS/
│     └── one_page_website.css
│── JS/
│     └── one_page_website.js




## 🖼️ Lightbox Feature

This project includes a JavaScript-powered lightbox/modal image gallery that allows users to click on images to view them in full-screen mode.
